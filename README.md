# quantum-harmonic-decision-model

Quantum Harmonic Decision Model
===============================

This repository contains the implementation and experimental data accompanying the paper:

"Quantum-Inspired Harmonic Decision Models: A Computational Framework for Music Generation and Cognitive Exploration"

The project explores harmonic generation as a structured decision-making problem using a quantum-inspired computational approach.


Overview
--------

The proposed framework models harmonization as a search process in a combinatorial harmonic space. It consists of two main components:

1. Quantum-inspired harmonizer
   - Interference-based search mechanism
   - Explores multiple harmonic candidates simultaneously
   - Inspired by QAOA-like optimization principles

2. Classical optimizer
   - Refines harmonic sequences using tonal harmony rules
   - Reduces unnecessary chord changes
   - Improves voice leading and cadence structure

How to Run
----------

1. Prepare input melody (MusicXML format)

2. Run harmonizer:
   python3 main.py

3. Run optimizer:
   python3 musicxml_optimizer.py

4. Output:
   - New MusicXML file with harmonized chords
   - Bass/left-hand accompaniment


Experimental Data
-----------------

The repository includes:

- Raw harmonizations generated by the quantum-inspired model
- Optimized harmonizations after classical post-processing
- Metric tables used in the paper:
    - Chord density
    - Harmonic function distribution (T/S/D)
    - Seventh chord ratio
    - Melodic coverage
    - Average chord duration
    - Bass movement (jump size)

- Expert evaluation summaries


Notes on Methodology
--------------------

The model does NOT implement physical quantum computation.

Instead, it uses mathematical concepts inspired by quantum theory:
- superposition (multiple candidate solutions)
- interference (selection mechanism)
- amplitude-based evaluation

The approach should be understood as:
"quantum-inspired computational modeling of decision processes"


Reproducibility
---------------

All results reported in the paper can be reproduced using the provided:
